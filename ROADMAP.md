# Open Bible Roadmap

> High-level plan for Open Bible — what's coming next and what's ahead.

---

## 🚀 v0.9 — Next

**Target:** August 2026

### Improvements
- [ ] Workspace performance and stability with multiple tabs
- [ ] Synchronized scrolling across grid panels
- [ ] Customizable keyboard shortcuts
- [ ] Mobile UI refinements (gestures, bottom sheet)
- [ ] Tab navigation history

### Infrastructure
- [ ] E2E tests with Playwright
- [ ] Flatpak and Tauri build polish
- [ ] Automated release pipeline

---

## 📅 v1.0 — First Stable Release

**Target:** October 2026

### Highlights
- **New Highlights & Notes experience** — full interface to mark, categorize, and organize highlights and annotations by verse
- **General UI polish** — visual refinement across all screens, design system consistency, micro-animations and transitions

### Features
- [ ] Highlights UI with categories and colors
- [ ] Rich verse note editor (basic formatting)
- [ ] Notes and highlights management (list, edit, delete)
- [ ] Export notes and highlights
- [ ] Design system refinement (typography, spacing, colors)
- [ ] Consistent transitions and micro-animations
- [ ] Polished dark mode
- [ ] Native desktop installers (DMG, MSI)
- [ ] Flatpak publishing on Flathub

- Dark mode
- Offline-first PWA (SQLite WASM + OPFS)

- Flatpak CI build
- Release notes and PWA update popup
- Optimized mobile navigation
- iOS-style mobile settings
- Close all tabs and new tab button
- Semantic releases with changelog
