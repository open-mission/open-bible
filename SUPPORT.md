# Support

Obrigado por usar o Open Bible! Aqui estão os canais disponíveis para obter ajuda.

## 📖 Documentação

- [README](README.md) — visão geral, setup e scripts
- [CONTRIBUTING](CONTRIBUTING.md) — guia para contribuidores

## 💬 Community

- **GitHub Discussions** — para perguntas, ideias e discussões gerais:
  https://github.com/open-mission/open-bible/discussions

## 🐛 Bug Reports

Encontrou um bug? Abra uma issue usando o template apropriado:
https://github.com/open-mission/open-bible/issues/new?template=bug_report.md

## ✨ Feature Requests

Tem uma ideia para uma nova funcionalidade?
https://github.com/open-mission/open-bible/issues/new?template=feature_request.md

## 🔒 Security

Reporte vulnerabilidades de segurança de acordo com nossa [Security Policy](SECURITY.md):
https://github.com/open-mission/open-bible/security/advisories/new

## ⏰ Response Times

- Issues: geralmente respondidas dentro de 48h
- Security reports: acknowledged within 48h, fix timeline per SECURITY.md
- Discussions: within 72h
