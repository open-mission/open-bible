# Support

Thank you for using Open Bible! Here are the available channels for getting help.

## 📖 Documentation

- [README](README.md) — overview, setup, and scripts
- [CONTRIBUTING](CONTRIBUTING.md) — guide for contributors

## 💬 Community

- **GitHub Discussions** — for questions, ideas, and general discussions:
  https://github.com/open-mission/open-bible/discussions

## 🐛 Bug Reports

Found a bug? Open an issue using the appropriate template:
https://github.com/open-mission/open-bible/issues/new?template=bug_report.md

## ✨ Feature Requests

Have an idea for a new feature?
https://github.com/open-mission/open-bible/issues/new?template=feature_request.md

## 🔒 Security

Report security vulnerabilities according to our [Security Policy](SECURITY.md):
https://github.com/open-mission/open-bible/security/advisories/new

## ⏰ Response Times

- Issues: generally responded to within 48h
- Security reports: acknowledged within 48h, fix timeline per SECURITY.md
- Discussions: within 72h
